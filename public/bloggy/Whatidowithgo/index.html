What do I do with Go?.

I usually build servers. I enjoy the nice API that it has. A good example is the http package The ListenAndServe method is really where the server begins. 

http.HandleFunc("/hello", helloHandler)
	log.Fatal(http.ListenAndServe(":8080", nil))
The HandleFunc is how routing is setup. I think a lot of people dont realize how easy this is to read. They just presume this is how go does it and dont stop to realize how simple this is.
